# 📊 Retail Sales Data Analysis

## 🎯 Project Objective
The purpose of this project is to load, clean, explore, and analyze retail sales data using Python (Pandas and Matplotlib). It demonstrates foundational skills in data analysis and provides initial business insights.

## 🛠️ Technologies Used
- **Python**
- **Pandas** (Data manipulation and cleaning)
- **Matplotlib** (Data visualization)

## 📂 Repository Contents
- `sales_data_sample.csv` - Original raw dataset.
- `Murad-1-sales_data_analysis.ipynb` - Google Colab/Jupyter Notebook containing data cleaning and visualization steps.
- `cleaned_data.csv` - Cleaned dataset ready for analysis.
- `note.md` - Documentation of data cleaning steps and initial business insights.

## 🚀 Key Insights
- The **highest number of orders** was recorded in 2004 (1351 orders).
- **92.6% of all orders are marked as "Shipped"**, indicating an efficient delivery process.
- The top countries by number of orders include the **USA, Spain, and France**.
